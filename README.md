A mini Amazon-style e-commerce web application built using **HTML, CSS, JavaScript**, and a simple **backend** for fetching product data and storing orders.  
This project is a personal practice clone focused on UI, interactivity, and basic end-to-end flow of an online shopping experience.

---

## âœ¨ Features

- **Home Page**
  - Displays a list of products with images, prices, and details
  - **Add to Cart** button for each product
  - Dynamic cart item count

- **Cart & Checkout**
  - Cart page with selected products and quantities
  - **Order summary** with subtotal, tax, and total price
  - **Payment summary / confirmation** section

- **Orders Page**
  - List of past orders
  - Shows basic **order details** and **status**

- **Tracking Page**
  - Simple UI to track an order
  - Displays **current status** of delivery

- **Backend**
  - API to fetch products
  - Stores orders / cart details
  - Can be extended to use any backend (Node, JSON server, etc.)
